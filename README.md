# clonepkg

Clone AUR pkgs with `git`.


## Usage

Clone clonepkg:

```sh
$ git clone https://github.com/atweiden/clonepkg && cd clonepkg
$ export PERL6LIB=lib
```

Clone pkgs:

```sh
$ ./bin/clonepkg \
    perl6-debugger-ui-commandline \
    perl6-terminal-ansicolor \
    perl6-grammar-debugger
```


## Installation

### Dependencies

- Rakudo Perl6
